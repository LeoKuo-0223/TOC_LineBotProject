## 功能
* 提供英日、英英兩種字典模式!

* 除了輸入欲查詢的單字外，只要透過按鈕快速回覆即可完整使用字典所有功能，節省下打字的時間 !

### 英英字典
1. 英文單字詞性、解釋
2. 多種發音以及音標
3. 範例句型
### 英日字典
1. 英文單字翻譯日文
2. 翻譯包含平假名、片假名、漢字以及發音
3. 詞性標註、解釋
4. 提供英、日文範例句型
